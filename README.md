# Audrey Ekstrom — Portfolio

## Tech Stack

- **React 19** + **Vite 8**
- Vanilla CSS with CSS variables for theming
- Custom font: Scratchy Lemon
- Deployed via **GitHub Pages** (`gh-pages`)

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview

# Deploy to GitHub Pages
npm run deploy
```

## Project Structure

```
src/
├── App.jsx          # Main application component
├── App.css          # Component and layout styles
├── index.css        # Global styles and CSS variables
└── assets/
    └── avatar.png   # Avatar image
```
